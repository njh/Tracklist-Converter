Tracklist Converter
===================

A web based tool to convert track lists between various formats.


Formats
-------

Support for the following formats is planned:

* Simple Text (artist - title)
* Fielded Text (key: value)
* [CUE]
* [haudio]
* [JSPF]
* [Music Ontology RDF]
* [XSPF]
